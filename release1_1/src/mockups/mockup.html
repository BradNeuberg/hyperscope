This is an HTML mockup for Phase 1 of the HyperScope development. If done properly, these mockups can be used as the basis for the actual HyperScope UI.
1A1
The blue bar above is the toolbar. There are two pulldown menus there, one for jumping and one for viewspecs.
1A2
This area is the document viewing area. Purple numbers -- when turned on -- are displayed on the right-hand margin. There are some controls embedded here as well. When you hover over a node, controls will appear in the left margin. These are viewspec shortcuts.
