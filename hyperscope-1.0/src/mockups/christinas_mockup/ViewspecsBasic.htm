Set View Detail (Power Users):

  1. Power Users menu button is depressed and Set Viewspecs command has been selected OR the user did a shortcut "sv" for Set Viewspecs
  2. Assumes this user is more advanced, has already memorized the most-used viewspecs, and wants the lean and mean prompter
  3. The More button on this overlay then morphs into the All Viewspecs overlay which is the works
  4. Button details
    1. ?: gives a popup help window describing Set Viewspecs command and how to use it (possibly more advanced description than offered via the Set View button
    2. More: swaps in the All Viewspecs overlay which includes a Less button in place of the More button
    3. OK: executes the command
    4. Cancel: aborts the command
    5. Reset: clears the viewspecs
