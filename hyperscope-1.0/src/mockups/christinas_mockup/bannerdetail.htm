Banner Detail:

  1. Color - going for a softer blue, closer to a copenhagen blue from Doug's scandanavian ancestry
  2. "hyperScope" going for a friendly and dynamic look, I like starting off lowercase (the blurring is unintentional)
  3. I moved Help button to the right since that is more typical placement for Help menus
  4. I used "Set View" instead of "Viewspecs" to keep the verb in there (this implicitly trains user Verb-Noun syntax) and conforms to what will be offered later in the Power Users menu and upwardly compatible with Turbo mode HOWEVER if you dropped the verb to simplify the direct command typein feature then I prefer to label this button "Views" instead of "Viewspecs" because the latter is I think overly technical for the rank beginner newcomer
  5. Added Power Users menu for LATER (implement last) listing all the commands offered, set apart from the 2 buttons rank beginners will use
  6. Display Hover Help (displayed temporarily in a text bubble when mouse hovers over this screen element):
    1. Jump: "Jump to a specific part of this file"
    2. Set View: "Change your view of this file"
    3. Help: "HyperScope Help"
    4. Power Users: "Menu of HyperScope commands for more advanced users"
    5. Icon thingy: "Copy Location capturing your current view"
